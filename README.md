# YandexGPT Broker

**AI-бот, который помогает покупателям с выбором блюд в ресторане.
Также помощник для подсчета каллорий в блюде.**

**Интеграция с Java-проектом.**

**Бот создан на Python 3.14 (FastAPI)**


## Быстрый старт

```bash
# 1. Скачать зависимости
pip install -r requirements.txt

# 2. Вставить API ключ YandexGPT и ID (получить на yandex.cloud/ru)
YANDEX_FOLDER_ID=""
YANDEX_API_KEY=""

# 3. Запустить
uvicorn main:app --reload
```

## Архитектура

`Сайт (сделан на Java)` → `FastAPI` → `YandexGPT API` → `PostgreSQL (история)`

## Ошибки API

| HTTP код | Описание |
|----------|----------|
| 400 | Пустое сообщение или некорректные параметры |
| 404 | Диалог не найден |
| 422 | Ошибка валидации JSON |
| 500 | Ошибка YandexGPT API или таймаут |

## Эндпоинты
POST /api/chat

Основной эндпоинт для общения с AI-помощником. 
Бот знает меню ресторана, может посоветовать блюдо по вкусу или подсчитать калории.

Запрос:

```json
{
  "message": "Какой состав блюда Noname?",
  "conversation_id": "uuid",
  "temperature": 0.3,
  "max_tokens": 300
}
```
| Поле              | Тип     | Описание                                                    |
|-------------------|---------|-------------------------------------------------------------|
| message           | string  | Текст запроса от пользователя                               |
| conversation_id   | string  | UUID диалога           |
| temperature       | float   | Креативность ответа (0.0–1.0)                               |
| max_tokens        | integer | Максимальная длина ответа      

Пример ответа:

```json
{
  "reply": "Noname — это ролл с лососем. Состав: нори (морская капуста), рис для суши, филе лосося (сякэ), огурец,",
  "сливочный сыр. Подаётся с васаби, имбирём и соевым соусом. Калорийность на 1 порцию (8 кусочков) — примерно 320-350 ккал. ",
  "Белки — 15 г, жиры — 12 г, углеводы — 38 г. Подойдёт для лёгкого обеда или перекуса.",
  "conversation_id": "uuid",
  "tokens_used": 78,
  "timestamp": "2026-04-04T16:20:15"
}
```

| Поле              | Что означает                                                          |
|-------------------|-----------------------------------------------------------------------|
| reply             | Текст ответа от AI-помощника                                          |
| conversation_id   | UUID диалога         |
| tokens_used       | Сколько токенов потрачено         |
| timestamp         | Время ответа                                                          |

Вопросы-примеры для бота:

```text
"Что посоветуешь из горячего?"

"У меня аллергия на орехи, что можно заказать?"

"Сколько калорий в пицце Noname?"

"Какое блюдо самое низкокалорийное?"
```
